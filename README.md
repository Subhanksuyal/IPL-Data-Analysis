# IPL Data Analysis using Python

This project analyzes 1169 IPL matches to understand:

- Impact of toss on match results
- Defending vs chasing trends
- Toss decision distribution
- Season-wise match growth

Key Insights:
- Toss winner wins ~51.58% matches
- Chasing teams dominate (615 vs 531)
- IPL remains competitive overall

Tools Used:
- Python
- Pandas
- Matplotlib
